Imagens

Para utilizarmos uma imagem, podemos pegar a partir de um link ou da pasta do seu computador.

Imagens a partir de Links

Para uma imagem a partir de um link, podemos usar a tag <img>. Ela possui três atributos principais, o primeiro é o src que define a origem da imagem, o segundo é o width, que vai definir a largura da sua imagem e o terceiro é o height que vai definir a altura da sua imagem.

Imagens Flexíveis

As imagens flexíveis nos ajudam a deixar o nosso site mais responsivo e compátivel com diversas telas. Para usar elas, temos algumas tags e atributos: A tag principal para isso é <picture> (Ela vai permitir você colocar mais de uma imagem para poder mudar o seu tamanho na hora de diminuir a tela). A segunda tag é a tag básica de imagem que você viu anteriormente. A próxima tag é a <source>, ela possui três atributos:O primeiro é o type que vai indicar o media type da imagem que estamos usando. O segundo é o srcset, esse atributo vai configurar o nome da imagem que será carregada quando o tamanho indicado for atingido e o terceiro é o media, ele indica o tamanho máximo( ou mínimo) a ser considerado para carregar a imagem indicada no atributo srcset.
