Emojis

Hoje em dia, os emojis são muito utilizados na mídia, para várias coisas, e em um site não seria diferente. Primeiro, você precisa do código do emoji que sempre começa com a letra U, exemplo: U+1F913. Para colocar um emoji, você deve antes substituir a parte do código do emoji U+ por um outro código: &#x e depois coloque o resto do código do emoji junto. Recomendo o site Emojipedia para os códigos.

Favicon

Favicon nada mais, nada menos é aquele ícone na parte da aba de algum site, na parte superior do navegador. Para usar um Favicon no seu site, você precisa ter o arquivo do ícone, que geralmente está no formato ICO.Os formatos suportados para favicon são, ICO, PNG e SVG.a tag para o favicon é <link>, quando você for escrever ela no VSirá aparecer vários tipos de link, escolha o que está escrito favicon na frente.

Listas

As listas estão presentes em muitos lugares, seja listas de desejos, de compras ou até de regras. Na internet não seria diferente, existem dois tipos de listas, as ordenadas e não ordenadas.

Listas Ordenadas

Para criar uma lista ordenada, usamos a tag <ol> e dentro dessa tag colocamos itens de lista(as tags <li>). Esse tipo de lista contém marcadores organizados em ordem, temos cinco tipos de marcadores:

  1. Listas numeradas
  1. Listas alfabéticas em maiúsculas
  1. Listas alfabéticas em minúsculas
  1. Listas com algarismos romanos em maiúsculas
  1. Listas com algarismos romanos em minúsculas

Para mudar o estilo da lista, basta colocar type dentro da tag

Listas Não Ordenadas

Para criar uma lista não ordenada, usamos a tag <ul> e dentro dessa tag colocamos itens de lista(as tags <li>). Esse tipo de lista contém marcadores desorganizados. Temos três tipos de marcadores:

  • Listas com uma bola preta pintada
  • Listas com uma bola com borda preta
  • Listas com um quadrado preto pintado
