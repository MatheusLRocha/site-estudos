Teoria das Cores

As cores podem influenciar no tempo em que o visitante do seu site passa nele. As cores passam emoções para o nosso subconsciente, mesmo que não seja feito de forma consciente. Percebemos as cores e sentimos a sua emoção mesmo sem ter a plena certeza de que alguém esteja usando a psicologia das cores.

Círculo Cromático

O círculo cromático nos ajuda a separar cores para a harmonização entre elas para saber se as cores escolhidas para nosso site vão fazer um sentido harmônico.

Temperatura e Harmonia

As cores podem ser divididas em temperatura: Quente e Frio. Assim como também podem ser divididas por esquemas harmônicos...

Representação de Cores

Para representar uma cor em CSS, podemos usar a forma rgb, hexadecimal e HSL. As cores de uma tela são compostas por três cores, r(red), g(green) e b(blue), se usarmos todas essas três cores primárias ao máximo obtemos o branco, se usarmos no mínimo obtemos o preto. Cada cor primária pode ter um valor entre 0 e 255, totalizando em 256 possibilidades para cada elemento. Um exemplo é a cor Teal. Seu código rgb(0, 171, 169) indica que existe uma quantidade 0 de vermelho, 171 de verde e 169 de azul. No código de cores hexadecimal(iniciando sempre com #) indica que 00 é a quantidade de vermelho, AB é a quantidade de verde e A9 é a quantidade de Azul. Essa cor pode também ser representada em CSS com outro formato baseado na maneira como o olho humano enxerga as cores: o padrão HSL. A função hsl(179,100%,34%) indica que temos 179 de hue(matiz), 100% de saturation(saturação) e 34% de luminosity/lightness(luminância).

Gradientes

Para usar um gradiente em CSS, devemos usar a tag background-image:linear-gradient(90deg,yellow,red). Um gradiente é considerado pelo navegador como se fosse uma imagem, por isso usamos a propriedade background-image na declaração CSS. O primeiro parâmetro da função indica o ângulo de inclinação de 90 graus e as seguintes indicam as cores do degradê. Podemos também gerar os gradientes radiais: background-image:radial-gradient(circle,red,yellow,green).
