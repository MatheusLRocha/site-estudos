Variáveis em CSS

Utilizamos as varíaveis em CSS para cadastrar os esquemas de cores e fontes do nosso site e isso vai facilitar muito na hora de efetuar eventuais mudanças estéticas no nosso site.

Para criarmos variáveis para nossas configurações devemos definir uma área de definições dentro do seu estilo para uma pseudo-classe chamada :root, que definem as configurações para a raiz de uma árvore, que vai servir para o documento inteiro.

Exemplo

*:root{ --cor1: #223075; --cor2: #91A2F7; --cor3: #4662F5; --cor4: #454D75; --cor5: #384FC2; --fonte-padrao: Arial, Helvetica, sans-serif; --fonte-titulo-especial: 'Teko', sans-serif; --fonte-titulo: 'Bebas Neue', sans-serif; }*

A partir de agora, definir cores e fontes em nossos elementos HTML ficará extremamente mais fácil e personalizável, utilizando a função var().

*body{ font-family: var(--fonte-padrao); color: var(--cor2); background-color: var(--cor1); }

[OBS] as variáveis personalizadas em CSS devem ter seus nomes iniciando dois traços obrigatoriamente.

Seletor * em CSS

Existe um seletor especial das CSS que é o asterisco(*), ele tem uma função muito especial, pois basicamente ele aplica uma configuração padrão para todos os elementos no código HTML ao qual o estilo está sendo aplicado.

Esse seletor global também ajuda a eliminar eventuais margens que os navegadores(user agents) adicionam a alguns elementos.

Espaçamento entrelinhas em Textos

Outra configuração muito importante que podemos fazer oara textos muito longos é o espaçamento entre as linhas do nosso texto. Usando a propriedade line-height, podemos dizer qual é o tamanho do espaço entre uma linha e a outra do texto.

[OBS] O valor padrão na maioria dos navegadores é algo próximo ao 1.2em.

Sombras em Textos

Assim como as caixas, também podemos aplicar sombras aos textos com uma propriedade específica: text-shadow. A propriedade text-shadow também tem os mesmos quatro parâmetros que os da box-shadow.

Personalizando as listas

Vamos ver como personalizar mais as listas:

ul{list-style-type: '\2714\0020'; columns: 2; list-style-position: inside;}

A propriedade list-style-type: faz com que o marcador seja personalizado. O valor \2714 corresponde ao símbolo ✔. O valor \0020 corresponde a um espaço em branco(também pode ser \00A0).

A segunda declaração vai organizar a lista em duas colunas. O total da lista vai ser dividido em duas colunas de partes iguais(ou quase).

A última declaração vai fazer com que os marcadores sejam exibidos na parte interna da caixa que contém a lista(normalmente a caixa de uma lista não inclui os marcadores).

Vídeos mais flexíveis

Quando incorporamos um vídeo do YouTube, isso é feito através de uma tag <iframe> que já vem com as configurações de um tamanho fixo e precisamos alterar isso para que nossa interface possa se tornar mais responsiva e adaptar o tamanho do vídeo dinamicamente, principalmente para telas pequenas.

Para isso, vamos colocar o <iframe> de incorporação dentro de uma <div> para que o vídeo esteja limitado dentro de um contêiner.

Após colocar o vídeo dentro de uma div, vamos fazer configurações de estilo para os dois elementos:

div.video{position: relative; background-color: blue; height: 0px; margin-left: -20px; margin-right: -20px; margin-bottom: 15px; padding-bottom: 59%;}

div.video > p{position: absolute; top: 5%; left: 5%; width: 90%; height: 90%;}

O que mais importa aqui é entender o funcionamento da propriedade position, que é a única que não vimos até aqui. O valor padrão para essa propriedade de posicionamento é static, que mantém a hierarquia conforme estabelecido no documento HTML.

Na div, nós colocamos o valor como relative para que seja considerado o posicionamento atual do elemento de divisão e que ele se mantenha adaptável para o caso de alteração no tamanho do navegador.

Já dentro do iframe, nós usamos o posicionamento absolute para que a div - que é o seu contêiner - torne-se o ponto de partida para o posicionamento do frame. A partir daí podemos utilizar propriedades para configurar o deslocamento à esquerda(left) e ao topo(top) e seu tamanho em largura(width) e altura(height), todos em porcentagem de tela.

[EXTRA] A tag transition-duration: 1s; faz com que quando você passe o mouse sobre algo, demore 1 segundo de transição.
